# El Pollo Loco

A 2D side-scrolling browser game built with JavaScript, HTML, and CSS.

## About the Project

El Pollo Loco is a platform game where the player controls Pepe on his adventure through the Mexican desert. Along the way, players must collect coins and salsa bottles, defeat chickens, and prepare for the final battle against the Endboss.
The game was developed using object-oriented JavaScript without external game engines. It focuses on clean code structure, animations, collision detection, sound management, responsive design, and game state management.

## Features

* Object-oriented JavaScript architecture
* Responsive desktop and mobile controls
* Animated character, enemies, and boss
* Collectible coins and bottles
* Throwing mechanics with bottle projectiles
* Endboss with multiple behavior states
* Sound effects and background music
* Pause, game over, and win screens
* Touch controls for mobile devices

## Controls

### Desktop

| Action               | Key   |
| -------------------- | ----- |
| Move Left            | ←     |
| Move Right           | →     |
| Jump                 | Space |
| Throw Bottle         | D     |
| Start Game           | S     |
| Pause                | P     |
| Exit to Start Screen | ESC   |

### Mobile

* Left and right touch buttons for movement
* Jump button
* Throw button

## Technologies Used

* HTML5

## Installation

1. Clone the repository:
```bash
git clone <repository-url>
```
2. Open the project folder.
3. Start a local server (e.g. Live Server in VS Code).
4. Open the game in your browser.
